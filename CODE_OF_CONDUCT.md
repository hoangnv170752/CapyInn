# Code of Conduct

## Our Standard

We want this project to be welcoming, technically rigorous, and respectful.

Examples of behavior that contributes to a positive environment:

- being respectful in disagreement
- giving actionable technical feedback
- assuming good intent while staying precise
- focusing on the code and the product rather than the person

Examples of unacceptable behavior:

- harassment, personal attacks, or discrimination
- doxxing or publishing private information
- bad-faith trolling or repeated disruptive behavior
- sexualized language or unwanted attention

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests,
discussions, and other public or private communication channels tied to the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
and other contributions that violate this Code of Conduct.
